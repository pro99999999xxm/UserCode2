**Usage:**

文件:

	点击 run.bat 以启动程序
	点击 Main\LD-testrun.bat 以启动Lock Down Browser
	config.ini 整界面样式，绑键等
	Main\Updates.txt 查看更新内容

界面:

	"L"  载入文本文档
	"F"  打开搜索界面(输入&回车以搜索)
	"<"  上一个搜索结果
	">"  下一个搜索结果
	"E"  关闭程序
	"S"  全屏截图(默认路径为 "C:\a")
	"-"  缩小至2x2大小(点击像素按钮以恢复)

键位:

	"Ctrl+Shift+Z" 键入以缩小至2x2大小(点击像素按钮以恢复)
	"Ctrl+滚轮" 缩放字体
