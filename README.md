---

## Usage 使用说明

### 📁 文件
- **启动程序**：双击运行 `Oblique Orthographic Conduit.exe`
- **实战测试**：运行 `Misc/LD-testrun.bat`
- **配置文件**：`config.ini`  
  - 用于设置：界面样式、透明度、键位绑定等

---

## 🎛️ 界面操作

| 按键 | 功能 |
| --- | --- |
| `L` | 载入文本文档 |
| `F` | 打开搜索界面（输入后按回车搜索） |
| `<` | 上一个搜索结果 |
| `>` | 下一个搜索结果 |
| `E` | 关闭程序 |
| `S` | 全屏截图（默认保存路径：`C:\a`） |
| `-` | 缩小至 `2x2` 大小（点击像素按钮即可恢复） |

---

## ⌨️ 键位操作

| 操作 | 功能 |
| --- | --- |
| `Ctrl + Shift + Z` | 缩小至 `2x2` 大小（点击像素按钮即可恢复） |
| `Ctrl + 鼠标滚轮` | 缩放字体 |
